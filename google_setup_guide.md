# 구글 클라우드 연동 가이드

Render에서 데이터를 안전하게 저장하려면 구글 시트 연동 키(Service Account Key)가 필요합니다. 
약간 복잡할 수 있지만, 한 번만 설정하면 평생 무료로 안전하게 쓸 수 있습니다!

## 1단계: 프로젝트 만들기
1.  [구글 클라우드 콘솔](https://console.cloud.google.com/)에 접속합니다.
2.  상단 로고 옆의 **프로젝트 선택**을 클릭하고 **"새 프로젝트"**를 누릅니다.
3.  프로젝트 이름(예: `railway-dashboard`)을 넣고 **만들기**를 누릅니다.

## 2단계: API 활성화
1.  좌측 메뉴 아이콘(햄버거) -> **API 및 서비스** -> **라이브러리**로 이동합니다.
2.  `Google Sheets API`를 검색하고 클릭합니다.
3.  **사용** 버튼을 누릅니다.

## 3단계: 서비스 계정(Service Account) 만들기
1.  좌측 메뉴 -> **API 및 서비스** -> **사용자 인증 정보**로 이동합니다.
2.  상단 **+ 사용자 인증 정보 만들기** -> **서비스 계정**을 선택합니다.
3.  이름(예: `server-bot`)을 입력하고 **완료**를 누릅니다.
4.  생성된 서비스 계정의 이메일 주소(예: `server-bot@....iam.gserviceaccount.com`)를 **복사**해두세요. (4단계에서 필요)

## 4단계: 키 다운로드
1.  방금 만든 이메일 주소를 클릭해서 상세 페이지로 들어갑니다.
2.  상단 **키** 탭 -> **키 추가** -> **새 키 만들기**를 누릅니다.
3.  **JSON**을 선택하고 **만들기**를 누르면 파일이 다운로드됩니다.
4.  이 파일 이름을 **`credentials.json`**으로 변경합니다.
5.  이 파일을 `server/` 폴더 안에 넣어주세요.

## 5단계: 구글 시트 공유
1.  [구글 스프레드시트](https://sheets.google.com/)에서 새 시트를 하나 만듭니다. (이름: `Railway DB` 추천)
2.  주소창의 URL에서 `/d/` 와 `/edit` 사이의 **긴 문자열(Sheet ID)**을 복사해두세요.
    - 예: `https://docs.google.com/spreadsheets/d/`**1aBcDeFgHiJkLmNoPqRsTuVwKyZ**`/edit`
3.  우측 상단 **공유** 버튼을 누르고, 아까(3단계) 복사한 **서비스 계정 이메일**을 초대합니다. (편집자 권한)
